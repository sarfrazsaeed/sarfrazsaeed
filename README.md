### ⚡ SmartHire v3 — HR & Job Matching System
> Commercial-grade HR candidate matching system built with Vite + React 18 + TypeScript

[![SmartHire](https://img.shields.io/badge/Live_Demo-10B981?style=for-the-badge)](https://sarfrazsaeed.github.io/smart-hr-job-matching/)
[![GitHub](https://img.shields.io/badge/Source_Code-181717?style=for-the-badge&logo=github)](https://github.com/sarfrazsaeed/smart-hr-job-matching)
[![Tests](https://img.shields.io/badge/Tests-36_Passing-10B981?style=for-the-badge)](https://github.com/sarfrazsaeed/smart-hr-job-matching/actions)

**Key Features:**
- 🎯 Weighted scoring algorithm — 70% skills · 20% experience · 10% education
- 📄 Client-side PDF resume parsing with auto skill extraction
- 💰 Pricing page with Monthly/Yearly toggle and USD/PKR currency switch
- 📊 Chart.js dashboard with skill frequency, job types, experience analytics
- 🧪 36 Vitest unit tests · GitHub Actions CI/CD · Auto-deploy to GitHub Pages
- 🏗 Built with: `React 18` `TypeScript` `Vite` `Tailwind CSS` `Framer Motion` `Chart.js` `React Router v6`
